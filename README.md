
Pour obtenir les executables, faites make dans le terminal en étant dans le même répertoire que le fichier Makefile avec les fichiers "projet.ml" et "test.ml" présent aussi dans ce répertoire.
Vous obtiendrez un fichier : test qui permet de tester si les fonctions sont valides dans les cas étudiés par mes tests.
Faites ./test pour executer ce fichier.
